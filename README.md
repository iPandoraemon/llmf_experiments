# 《大模型基础》课程实验

本项目包含《大模型基础》课程的所有实验代码和报告。

## 项目结构

每个实验都有独立的文件夹，包含以下内容：
- `README.md` - 实验说明文档（环境搭建、运行步骤等）
- `requirements.txt` - Python 依赖包列表
- `*.py` - 实验代码文件
- `实验报告_学号-姓名.md` - 实验报告模板

## 实验列表

| 实验编号 | 实验名称 | 文件夹 | 状态 |
|---------|---------|--------|------|
| 实验1 | 本地部署与API调用 | [exp_private_deploy](./exp_private_deploy/) | ✅ |
| 实验2 | 大模型的文本情感分类 | [exp_prompt_emotion_cls](./exp_prompt_emotion_cls/) | ✅ |

## 使用说明

1. 进入对应的实验文件夹
2. 按照该实验的 `README.md` 说明搭建环境
3. 根据实验要求，实现代码并运行
4. 分析实验结果
5. 撰写实验报告

## 实施建议

1. 每一个实验独立文件夹中创建一个虚拟环境
2. 严格按照模板中所提及的要求，完成实验报告
3. 按要求提交实验报告和相应的结果文件
